## Gym-Member-Churn-Analysis


## Objective
Analyze gym customer data to identify churn patterns and retention opportunities using Excel.


## Tools Used
- Microsoft Excel
  - Power Query
  - Pivot Tables
  - Pivot Charts
  - Slicers
 
  
## Key KPIs
- Total Customers
- Membership Type Distribution
- Average Calories burnt by a customer

  
## Key Insights
- Monthly membership customers show higher churn risk
- Members aged 38–47 have the highest average tenure
- Reduced visit frequency strongly correlates with churn
- Overall churn rate indicates scope for retention strategies

## Files
- `GYM_Members_RAW-Dataset.csv` – Raw Gym Dataset
- `Gym hurn Analysis.xlsx` – Interactive Excel dashboard  
- `Gym Member Churn Analysis Dashboard.pdf` – Dashboard snapshot for quick view
